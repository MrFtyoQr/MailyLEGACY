DESCRIPCIÓN
--------------------------------------------------------------------------------
Módulo de analítica y dashboard. Agrega datos de salud del paciente para
presentar un resumen ejecutivo, calcular adherencia a medicamentos y generar
insights inteligentes usando IA. El modelo de IA varía según el plan del
paciente (M09 Payments):

  FREE    → OpenAI gpt-4o-mini  (insights básicos)
  SILVER  → OpenAI gpt-4o       (insights estándar)
  GOLD    → OpenAI gpt-4o       (insights detallados)
  PLATINUM→ Claude claude-sonnet-4-6 (análisis clínico avanzado)

--------------------------------------------------------------------------------
MODELOS (apps/analytics/models.py)
--------------------------------------------------------------------------------
AdherenceReport → snapshot semanal/mensual de adherencia por paciente
HealthInsight   → insight generado por IA, asociado a un paciente

Tipos de Insight (InsightType):
  MEDICATION_ADHERENCE  → análisis de adherencia a medicamentos
  VITAL_TREND           → tendencia de signos vitales
  LAB_ANALYSIS          → análisis de resultados de laboratorio
  GENERAL_HEALTH        → resumen general de salud

ENDPOINTS EXPUESTOS
--------------------------------------------------------------------------------
Paciente:
  GET  /api/v1/analytics/dashboard/          → resumen ejecutivo del paciente
  GET  /api/v1/analytics/adherence/          → adherencia (?days=7|30|90)
  GET  /api/v1/analytics/insights/           → insights generados
  POST /api/v1/analytics/insights/generate/  → solicitar nuevo insight

Doctor:
  GET  /api/v1/analytics/patient/{id}/dashboard/  → dashboard del paciente

--------------------------------------------------------------------------------
TAREAS CELERY

generate_weekly_adherence_report(patient_id)  → cada lunes, snapshot de la semana
generate_patient_insights(patient_id)         → semanal, genera insights con IA

--------------------------------------------------------------------------------
DECISIONES DE DISEÑO
--------------------------------------------------------------------------------
1. AdherenceReport es un snapshot periódico — no se recalcula en tiempo real.
2. HealthInsight tiene TTL implícito: se genera uno nuevo por semana; los viejos
   quedan en historial.
3. El dashboard GET es síncrono y agrega datos en memoria (sin modelo extra).
4. La IA recibe contexto anonimizado: sin nombres, solo métricas numéricas.
5. Los costos de IA solo aplican a usuarios con plan activo SILVER+; FREE recibe
   insights simplificados sin llamada externa.
6. Prompt caching activado para Claude (Anthropic SDK) para reducir costos en
   consultas frecuentes.
