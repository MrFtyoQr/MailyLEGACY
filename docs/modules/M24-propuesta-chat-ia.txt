MODELOS NUEVOS NECESARIOS (Django)
--------------------------------------------------------------------------------

class AIConversation(models.Model):
    patient       = models.ForeignKey(PatientProfile, on_delete=models.CASCADE)
    created_at    = models.DateTimeField(auto_now_add=True)
    last_message  = models.DateTimeField(auto_now=True)
    message_count = models.PositiveIntegerField(default=0)

    class Meta:
        ordering = ['-last_message']

class AIChatMessage(models.Model):
    ROLE_CHOICES = [('user', 'Usuario'), ('assistant', 'Asistente')]

    conversation  = models.ForeignKey(AIConversation, on_delete=models.CASCADE,
                                      related_name='messages')
    role          = models.CharField(max_length=10, choices=ROLE_CHOICES)
    content       = models.TextField()
    is_voice      = models.BooleanField(default=False)   # enviado por voz
    provider      = models.CharField(max_length=20, blank=True)  # OPENAI | ANTHROPIC
    tokens_used   = models.PositiveIntegerField(default=0)
    created_at    = models.DateTimeField(auto_now_add=True)

    class Meta:
        ordering = ['created_at']

--------------------------------------------------------------------------------
ENDPOINTS PROPUESTOS
--------------------------------------------------------------------------------

POST /api/v1/ai/chat/
  Auth:    Bearer JWT
  Body:    { "message": string, "conversation_id": uuid | null }
  Returns: {
    "conversation_id": uuid,
    "response": string,
    "provider": "OPENAI" | "ANTHROPIC",
    "disclaimer": "Este mensaje es orientativo y no constituye diagnóstico médico."
  }
  Rate limit: 20/hora para FREE, 100/hora para PLATINUM

GET /api/v1/ai/chat/history/
  Returns: lista de conversaciones con último mensaje

GET /api/v1/ai/chat/{conversation_id}/messages/
  Returns: historial de mensajes de la conversación

SISTEMA DE PRIVACIDAD Y ANONIMIZACIÓN
--------------------------------------------------------------------------------

Antes de enviar a OpenAI/Anthropic, el contexto del paciente se anonimiza:

DATOS QUE SÍ SE ENVÍAN (numéricos, sin identificadores):
  - Últimos 3 valores de signos vitales (valor + fecha relativa)
  - Lista de tipos de medicamentos (sin nombres de marca)
  - Edad en rangos (ej. "adulto 30-40")
  - Resumen de adherencia (%)

DATOS QUE NUNCA SE ENVÍAN:
  - Nombre, apellido, email, CURP, RFC
  - Diagnósticos en texto libre
  - Notas del médico
  - ID de paciente
  - Dirección, teléfono

7. Máximo 150 palabras por respuesta."
